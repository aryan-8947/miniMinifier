Simple compression (128 bytes)

  • Removes unnecessary whitespaces
  • Removes trailing spaces and slashes in tags
  • Removes unnecessary closing tags
  • Doesn't remove HTML comments because they can be used by templating engines 
<textarea oninput='for(i=9;i--;)t.value=t.value.replace(/<\/[tl].>\s*|[\s/]*(^|$|>)\s*|(\s)\s|<\/p>\s*(?=<p>|$)/gi,"$1$2")'id=t>



More compression (136 bytes)

  • Removes unnecessary whitespaces
  • Removes trailing spaces and slashes in tags
  • Removes unnecessary closing tags
  • Remove HTML comments 
<textarea oninput='for(i=9;i--;)u.value=u.value.replace(/<!.*?->|<\/[tl].>\s*|[\s/]*(^|$|>)\s*|(\s)\s|<\/p>\s*(?=<p>|$)/gi,"$1$2")'id=u>
